    Evaluator API¶

    The evaluator module provides tools for assessing the quality of intertextual detection.

    IntertextEvaluator¶

    The main class for evaluating detection results against ground truth.

    Constructor¶

    IntertextEvaluator(
    predictions: List[Dict[str, Any]],
    ground_truth: List[Dict[str, Any]],
    query_id_key: str = "query_id",
    reference_id_key: str = "reference_id",
    label_key: str = "label",
    score_key: str = "score"
)

    Parameters¶

    Parameter Type Default Description
    predictions List[Dict] - List of prediction dictionaries
    ground_truth List[Dict] - List of ground truth dictionaries
    query_id_key str "query_id" Key for query ID in dictionaries
    reference_id_key str "reference_id" Key for reference ID
    label_key str "label" Key for ground truth labels
    score_key str "score" Key for prediction scores

    Methods¶

    evaluate()¶

    Compute all evaluation metrics.

    def evaluate(threshold: float = 0.5) -> Dict[str, float]

    Parameters:

    Parameter Type Default Description
    threshold float 0.5 Classification threshold

    Returns: Dictionary with all computed metrics.

    Example:

    metrics = evaluator.evaluate(threshold=0.6)
print(metrics)
# {
#     'precision': 0.85,
#     'recall': 0.72,
#     'f1': 0.78,
#     'accuracy': 0.89,
#     'mrr': 0.65,
#     'map': 0.58,
#     ...
# }

    Example¶

    from locisimiles import IntertextEvaluator

predictions = [
    {"query_id": "1", "reference_id": "a", "score": 0.9},
    {"query_id": "1", "reference_id": "b", "score": 0.7},
    {"query_id": "2", "reference_id": "c", "score": 0.85},
]

ground_truth = [
    {"query_id": "1", "reference_id": "a", "label": 1},
    {"query_id": "1", "reference_id": "b", "label": 0},
    {"query_id": "2", "reference_id": "c", "label": 1},
]

evaluator = IntertextEvaluator(predictions, ground_truth)

# Get all metrics
metrics = evaluator.evaluate()

# Or compute individual metrics
print(f"Precision: {evaluator.precision():.3f}")
print(f"Recall: {evaluator.recall():.3f}")
print(f"F1: {evaluator.f1_score():.3f}")
print(f"MRR: {evaluator.mrr():.3f}")

    Example¶

    from locisimiles.evaluator import compute_precision, compute_recall, compute_f1

y_true = [1, 0, 1, 1, 0]
y_pred = [1, 0, 0, 1, 0]

precision = compute_precision(y_true, y_pred)  # 1.0
recall = compute_recall(y_true, y_pred)        # 0.667
f1 = compute_f1(y_true, y_pred)                # 0.8

    Ground Truth Format¶

    Ground truth data should be provided as a list of dictionaries with the following structure:

    [
    {
        "query_id": "verse_id_1",
        "reference_id": "target_verse_42",
        "label": 1  # 1 for match, 0 for non-match
    },
    ...
]

    Or loaded from a CSV file:

    query_id,reference_id,label
verse_id_1,target_verse_42,1
verse_id_1,target_verse_15,0
verse_id_2,target_verse_8,1

    The evaluator handles both positive and negative examples, allowing for comprehensive evaluation of both precision and recall.
