    Document API¶

    The document module provides classes for representing text collections.

    TextSegment¶

    A TextSegment represents an individual unit of text with an identifier.

    Constructor¶

    TextSegment(id: str, text: str, metadata: Optional[Dict[str, Any]] = None)

    Parameters¶

    Parameter Type Description
    id str Unique identifier for the segment
    text str The text content
    metadata Dict[str, Any] Optional metadata dictionary

    Properties¶

    Property Type Description
    id str The segment identifier
    text str The text content
    metadata Dict[str, Any] Metadata dictionary (defaults to empty dict)

    Methods¶

    to_dict()¶

    Convert the segment to a dictionary.

    def to_dict() -> Dict[str, Any]

    Returns: Dictionary with id, text, and metadata keys.

    Example¶

    from locisimiles import TextSegment

segment = TextSegment(
    id="aen.1.1",
    text="Arma virumque cano",
    metadata={"book": 1, "line": 1}
)

print(segment.id)       # "aen.1.1"
print(segment.text)     # "Arma virumque cano"
print(segment.metadata) # {"book": 1, "line": 1}

    Document¶

    A Document is a collection of TextSegment objects.

    Constructor¶

    Document(segments: List[TextSegment])

    Parameters¶

    Parameter Type Description
    segments List[TextSegment] List of text segments

    Properties¶

    Property Type Description
    segments List[TextSegment] The list of segments

    Methods¶

    from_csv()¶

    Load a document from a CSV file.

    @classmethod
def from_csv(
    path: str,
    id_column: str = "id",
    text_column: str = "text",
    metadata_columns: Optional[List[str]] = None,
    delimiter: str = ","
) -> Document

    Parameters:

    Parameter Type Default Description
    path str - Path to the CSV file
    id_column str "id" Name of the ID column
    text_column str "text" Name of the text column
    metadata_columns List[str] None Additional columns to include as metadata
    delimiter str "," CSV delimiter character

    Returns: Document instance.

    Example:

    doc = Document.from_csv(
    "vergil.csv",
    id_column="verse_id",
    text_column="latin_text",
    metadata_columns=["book", "line"]
)

    from_records()¶

    Create a document from a list of dictionaries.

    @classmethod
def from_records(
    records: List[Dict[str, Any]],
    id_key: str = "id",
    text_key: str = "text"
) -> Document

    Parameters:

    Parameter Type Default Description
    records List[Dict] - List of record dictionaries
    id_key str "id" Key for the ID field
    text_key str "text" Key for the text field

    Returns: Document instance.

    Example:

    records = [
    {"id": "1", "text": "First verse", "author": "Vergil"},
    {"id": "2", "text": "Second verse", "author": "Vergil"},
]
doc = Document.from_records(records)

    get_by_id()¶

    Retrieve a segment by its ID.

    def get_by_id(segment_id: str) -> Optional[TextSegment]

    Parameters:

    Parameter Type Description
    segment_id str The segment ID to look up

    Returns: TextSegment if found, None otherwise.

    texts()¶

    Get a list of all text content.

    def texts() -> List[str]

    Returns: List of text strings from all segments.

    ids()¶

    Get a list of all segment IDs.

    def ids() -> List[str]

    Returns: List of ID strings from all segments.

    __len__()¶

    Get the number of segments.

    len(document)  # Returns number of segments

    __iter__()¶

    Iterate over segments.

    for segment in document:
    print(segment.text)

    Example¶

    from locisimiles import Document, TextSegment

# Create from segments
doc = Document([
    TextSegment("1", "Arma virumque cano"),
    TextSegment("2", "Troiae qui primus ab oris"),
])

# Access properties
print(len(doc))          # 2
print(doc.texts())       # ["Arma virumque cano", "Troiae qui primus ab oris"]
print(doc.ids())         # ["1", "2"]

# Lookup by ID
segment = doc.get_by_id("1")
print(segment.text)      # "Arma virumque cano"
